# Project Title 
Enhanced Real Time-Drone Detection using Yolo

## Over view
This project is a real-time drone detection system developed using the YOLOv5 deep learning model. The system detects drones in images and video streams while distinguishing them from birds, improving surveillance accuracy and reducing false detections.

## Features
Real-time drone detection.
Drone vs Bird classification.
YOLOv5-based object detection.
Bounding box visualization with confidence scores.
High detection accuracy in different environments.
Real-time monitoring support.

## Technologies Used
Python
YOLOv5
PyTorch
OpenCV
NumPy
Google Colab
Roboflow

## Dataset
The model was trained using a combined dataset of drone and bird images. Data augmentation techniques such as image resizing, rotation, flipping, and brightness adjustment were applied to improve detection performance.

## Methodology
Data Collection and Annotation.
Dataset Splitting.
Image Preprocessing.
YOLOv5 Model Training.
Real-Time Object Detection.
Performance Evaluation.

## Output
The system detects drones in real-time and displays:
Bounding Boxes
Object Labels (Drone/Bird)
Confidence Scores

## Advantages
High detection accuracy.
Faster real-time detection.
Reduced false alarms.
Effective detection of small drones.
Easy integration with surveillance systems.

## Future Enhancements
